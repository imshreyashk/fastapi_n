# FastAPI Kafka App

A high-performance FastAPI application integrated with Apache Kafka for real-time data streaming.

## 🚀 Features
- **FastAPI**: Modern, fast web framework for building APIs.
- **Kafka Integration**: Asynchronous producer/consumer logic.
- **Uvicorn**: Lightning-fast ASGI server implementation.

## 🛠️ Tech Stack
- **Language:** Python 3.10+
- **Framework:** [FastAPI](https://fastapi.tiangolo.com)
(https://github.com)
- **Environment:** [uv](https://github.com) (Package Manager)
